
## $(date +%Y-%m-%d) - [Replace native titles with Tooltip components]
**Learning:** Native `title` attributes on icon-only buttons cause a sluggish OS-level tooltip to appear, making the UI feel clunky and unresponsive.
**Action:** Replace `title` attributes with the standard UI `Tooltip` component for immediate, beautifully styled visual feedback while maintaining `aria-label` attributes for screen readers.
